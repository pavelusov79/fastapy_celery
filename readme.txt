Приложение для отслеживания цен карточек товара с сайта wildberries.

Возможности проекта:
Автоматическое отслеживание динамики цен товаров с сайта wildberries.ru c заданным диапазоном дат и временным интервалом
(1 час, 12 часов, 24 часа).
Задачи коллекционирования данных реализованы в виде ассинхронных задач celery.

1. Регистрация пользователя (логин, пароль, почта)
2. Добавление, редактирование, удаление товаров для отслеживания динамики цен (наименование и артикул)
3. Постановка пользователем задачи отслеживания (выбирается товар, устанавливается дата окончания отслеживания,
выбирается интервал отслеживания. Редактирование задачи, удаление задачи.
4. Просмотр результатов отслеживания карточки товара по текущему пользователю.

Инструкции запуска приложения

1. Скачать проект с github
2. В папке проекта выполнить команду: "docker compose up", дождаться установки и запуска приложения.
3. Открыть приложение в браузере 0.0.0.0:8000/
4. Зарегистрировать нового пользователя 0.0.0.0:8000 перейдя на страницу регистрации
5. Авторизовать нового пользователя.
6. Добавить товар для отслеживания цен. Ввести название товара и его артикул, который указан в карточке товара на сайте wildberries.
Например, наименование: Зарядное устройство, артикул: 63208991
7. Добавить новую задачу перейдя в мои задачи и добавив новую.
При добавлении задачи выбрать из списка название товара для отслеживания цен, выбрать из выпадающего списка период отслеживания: 1 час, 12 часов, 24 часа. 
Для теста выберите 30 сек test. Выбрать дату окончания выполнения задачи. При создании задачи она автоматически запустится.
8. После добавления задачи запустится задача celery. Результат спарсенных цен товара можно увидеть перейдя на вкладку отслеживаемые товары.
Задачи можно удалять и редактировать. Если задача celery завершена то на вкладке мои задачи статус будет указан "неактивная". 
При редактрировании задачи в статус активная и если дата завергения парсинга будет болше чем текущее время то задача celery 
снова будет автоматически запущена.
